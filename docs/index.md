# Sobre

Aqui você encontrará toda a documentação de nossa api para integrar seu sistema ao Pedidos Digitais.

## Autenticação

Para criar um token e começar a usar a API, siga os passos abaixo:

- Acesse seu painel do Pedidos Digitais
- Acesse o menu **Configurações > API**
- Clique no botão **Novo Token +**
- Informe o nome (normalmente onde este token será usado)
- Clique no botão **Cadastrar**

Seu token será gerado e por segurança ele aparecerá na sua tela somente esta vez.
